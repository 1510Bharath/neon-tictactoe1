# ⚡ Neon Tic Tac Toe

A glowing neon-themed Tic Tac Toe game built with React.

## How to Run

```bash
npm install
npm start
```

Opens at http://localhost:3000

## Features
- 🌟 Neon glow effects — X in blue, O in red, purple grid
- 👥 2 Player mode
- 🤖 VS AI mode (minimax algorithm — unbeatable!)
- ✨ Particle confetti on win
- 📊 Score tracking across rounds
- 📱 Responsive design
- 💡 Animated background grid
- 🎆 Symbol pop-in animations
- 💫 Neon flicker title effect
